# Vite

Vite est un outil de build moderne pour le web, conçu pour offrir une experience de developpement ultra-rapide pour les applications front-end.

## Points cles

- **Demarrage instantane du serveur de developpement** : grâce a l'utilisation d'ESM natif, pas de bundling initial.
- **Hot Module Replacement (HMR) ultra-rapide** : les modifications de code sont refletees instantanement dans le navigateur.
- **Support natif de TypeScript, JSX, CSS, Vue, React, etc.**
- **Build optimise** : base sur Rollup, supporte le mode multi-pages et librairies.
- **Système de plugins flexible** : compatible avec les plugins Rollup et des options specifiques a Vite.
- **Support du SSR (Server-Side Rendering)** : facile a configurer.
- **Open source (licence MIT)**

## Installation rapide

```bash
npm create vite@latest
cd mon-projet
npm install
npm run dev
```

## Documentation

- Site officiel : https://vitejs.dev/

## Exemples d'utilisation

- Creer une application Vue, React, Svelte, etc. en quelques secondes.
- Developpement rapide grâce au rechargement instantane.
- Build de production optimise et rapide.
